HTTP method Path Action Used for
GET /photos index Display a list of all photos.
GET /photos/new new Return an HTML form for creating a new photo.
POST /photos insert Insert a new photo.
GET /photos/:id show Display a specific photo.
GET /photos/:id/edit edit Return an HTML form for editing a photo.
PUT /photos/:id update Update a specific photo.
GET /photos/:id/exclude exclude Return an HTML form for excluding a photo.
DELETE /photos/:id delete Delete a specific photo.
GET /photos/find find Return an HTML form for find a photo or more.
GET /photos/find?q=:val locate Find a photo or more.
